Nexus App

Offline-first convention planner built to help users browse events and manage personal schedules.

Overview

Nexus is a convention planning app designed to help users browse event schedules, build personal itineraries, and navigate convention spaces with an offline-first approach.

My Role

Designed and developed the app structure, event management flow, and user-focused planning features.

Core Features

  • Offline-first event browsing
  • Favorites and personal schedule building
  • Conflict awareness for overlapping events
  • Searchable tags and event organization
  • Planned venue maps and navigation support

Technical Focus

Focused on usability, local-first architecture, scalable event data handling, and a clean UI flow for convention attendees.

Current State

The project is under active development, with core planning and event management functionality established and additional UX and navigation features planned.
