Cosmeredle Solver

A client-side deduction tool designed to narrow Cosmere character candidates through structured data filtering, live UI updates, and multi-constraint search.

Stack: Vanilla JavaScript
Data Source: JSON
Type: Web Tool
Focus: Constraint-Based Filtering

Overview

Cosmeredle Solver is a browser-based deduction tool built to help users narrow down possible Cosmere character candidates in a puzzle-style format. Instead of manually tracking possibilities, the tool lets users apply multiple constraints at once and immediately see the filtered candidate pool update in real time.

My Role

I designed and implemented the filtering logic, interface structure, and state flow for the tool. The goal was to create a fast, readable system that could handle many simultaneous constraints without becoming difficult to use.

Core Features

  • Live Multi-Constraint Filtering: Characters can be narrowed by world, first appearance, species base, subtype, and abilities.
  • Positive and Negative Rules: Users can define both what a candidate must have and what they must not have.
  • Dynamic Control Population: Filter controls are populated directly from the dataset rather than hard-coded.
  • Dependent Filter Updating: Available options shrink as candidates are removed, keeping the UI aligned with valid possibilities.
  • Searchable Filter Lists: Large checkbox groups remain usable through built-in search inputs.
  • Selected Candidate Details: Clicking a result shows a structured detail panel for the selected character.

Technical Focus

The project centers on client-side data handling and UI-driven state updates. Character data is loaded from JSON, controls are generated from the dataset, and the filtering pipeline applies a sequence of inclusion, exclusion, and relational rules to produce the remaining candidate list.

A key technical focus was maintaining consistency between the result set and the available controls. As filters change, dependent controls update to reflect only possible remaining values while preserving any active selections already chosen by the user.

Design Challenges

The main challenge was balancing complexity and readability. The solver needed to support many overlapping filters while still making it clear why the candidate count was changing. This required careful handling of active filter visibility, reset behavior, search within filter groups, and selected-character feedback.

Development Status

Cosmeredle Solver is a completed utility-style project that demonstrates structured front-end logic, interactive filtering systems, and a focused data-driven interface for solving a niche but complex problem.
