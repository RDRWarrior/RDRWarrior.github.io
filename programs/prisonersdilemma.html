Prisoner’s Dilemma Simulator

Interactive program for testing strategy behavior and comparing outcomes in repeated Prisoner’s Dilemma scenarios.

Overview

This simulator allows users to select and compare Prisoner’s Dilemma strategies to see how they perform across repeated interactions.

My Role

Implemented the strategy logic, comparison structure, and user-facing interaction flow.

Technical Focus

Focused on simulation logic, outcome comparison, and structured program behavior.

Current State

The project demonstrates applied programming and simulation-oriented logic rather than game design.
