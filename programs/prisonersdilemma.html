Luthadel / Elendel Rendering Transition

An Advanced Graphics project that transforms Kredik Shaw’s ash-covered Luthadel into Elendel’s Field of Rebirth through a time-driven visual transition built with custom OpenGL rendering systems.

Overview

This project was built as an Advanced Graphics transition piece inspired by Mistborn. The core idea was to show the destruction of Kredik Shaw and the transformation of Luthadel into Elendel’s Field of Rebirth. The final sequence is not just a static scene swap, but a timed visual narrative in which architecture collapses, ash and mist effects reshape the atmosphere, and the environment transitions from an oppressive red-lit world into a brighter memorial landscape. :contentReference[oaicite:2]{index=2}

My Role

I designed and implemented the full transition experience, including the rendering pipeline, procedural scene construction, environmental effects, camera systems, and the sequential logic controlling the transformation from one era to the next.

Scene Structure

The project is built around two primary scene states. The first presents Luthadel under a red sun with drifting ash and Kredik Shaw dominating the skyline. The second shows Elendel’s Field of Rebirth, including a central memorial statue, ring-based park layout, and cleaner daylight atmosphere. The transition between them is time-driven and sequential rather than instantaneous. :contentReference[oaicite:3]{index=3}

Core Rendering Features

  • Custom Shader Pipeline: The project uses a custom OpenGL shader pipeline and scene graph system for rendering control and object organization.
  • Directional Lighting: Scene lighting includes ambient, diffuse, and specular components, with the sun color interpolating across the transition.
  • Environment Mapping: Dual cubemap skyboxes support distinct Luthadel and Elendel atmospheres.
  • Automatic Camera Track: A predefined camera spline path presents the scene cinematically until free camera mode is toggled.
  • Scene Graph Management: Scene nodes are added and removed in sequence as the transition progresses. :contentReference[oaicite:4]{index=4}

Advanced Technical Features

  • Real-Time Shadow Mapping: A dynamic shadow pass updates as collapsing and transforming geometry changes the scene. :contentReference[oaicite:5]{index=5}
  • Post-Processing: A custom framebuffer and transition shader handle full-screen blending and era-shift effects. :contentReference[oaicite:6]{index=6}
  • Particle Systems: The project includes both falling ash and a mist-wave system used during the transformation. :contentReference[oaicite:7]{index=7}
  • Procedural Geometry: Kredik Shaw, walls, park rings, canals, and other scene structures are created from simple primitives and runtime mesh generation.
  • Dynamic Lighting & Sky Blending: The project transitions smoothly from Luthadel’s red haze to Elendel’s clear daylight through lighting and skybox interpolation.
  • Camera Shake: Sinusoidal camera offsets are used during the collapse phase to create physical weight during destruction.

Material & Atmospheric Work

One of the strongest visual details in the project is the treatment of Kredik Shaw’s stained-glass windows. These use emissive materials combined with colored diffuse maps and low-frequency sine-based modulation to mimic flickering internal firelight. The project also uses alpha-blended ash particles drifting across the scene, fading out during the transition to symbolize the cleansing of the sky as the world changes from Luthadel to Elendel. :contentReference[oaicite:11]{index=11}

Performance & Structure

According to the project features sheet, the system was built to run smoothly at roughly 60 FPS in fullscreen while supporting multiple advanced rendering techniques at once. That combination of visual ambition and maintained runtime performance was an important technical goal of the project. :contentReference[oaicite:12]{index=12}

Controls

  • Space: Begin transition from Luthadel to Elendel
  • R: Reset to Luthadel
  • F: Toggle free camera
  • WASD: Move in free camera
  • F11: Toggle fullscreen

Outcome

This project is one of the clearest examples in my portfolio of combining technical graphics programming with thematic scene direction. It shows how rendering systems, particle effects, timing logic, shader work, and procedural construction can be orchestrated to support not just a pretty scene, but a visual transformation with narrative weight.
